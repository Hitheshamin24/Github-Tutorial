# 🧾 Basic Git Commands

## 📌 Check Status

```bash
git status
```

---

## 📌 Add Files

```bash
git add file.txt
git add .
```

---

## 📌 Commit Changes

```bash
git commit -m "message"
```

---

## 📌 View History

```bash
git log
```

---

## 📌 Check Differences

```bash
git diff
```

---

## 📌 Remove File

```bash
git rm file.txt
```

---

## 📌 Summary Table

| Command    | Purpose       |
| ---------- | ------------- |
| git init   | Start repo    |
| git add    | Stage changes |
| git commit | Save changes  |
| git status | Check status  |
| git log    | View history  |
