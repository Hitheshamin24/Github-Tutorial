# 📘 What is Git?

## 🔹 Definition

Git is a **version control system** that helps you track changes in your code.

---

##  Simple Explanation (Beginner Friendly)

Imagine you are writing notes 📒:

* You save version 1
* Then you edit → version 2
* Then again → version 3

Git helps you:

* Save all versions
* Go back anytime
* See what changed

---

##  Why Git is Important?

* Tracks code changes
* Prevents data loss
* Helps in teamwork
* Used in all software companies

---

##  Real-Life Example

Think of **Google Docs version history**:

* You can see edits
* You can restore old versions

 Git works the same way but for code.

---

## ⚙️ Key Features

* Version tracking
* Branching
* Collaboration
* Backup system
