# 🔄 Git Workflow

## 🔹 Basic Workflow

```
Working Directory → Staging Area → Repository
```

---

## 🧠 Simple Explanation

1. You edit files
2. Add them (`git add`)
3. Commit them (`git commit`)

---

## 🔁 Flow Example

```bash
git add .
git commit -m "Updated project"
```

---

## 📊 Workflow Diagram

```
Code → Add → Commit → Saved
```
