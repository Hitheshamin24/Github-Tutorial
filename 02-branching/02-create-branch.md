#  Creating and Switching Branches

##  Create Branch

```bash id="5y9l7p"
git branch feature-login
```

---

##  Switch Branch

```bash id="0l2w8v"
git checkout feature-login
```

---

```bash id="pq8m1x"
git checkout -b feature-login
```

 Creates + switches branch
