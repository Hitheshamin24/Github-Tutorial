# Merging Branches

## What is Merge?

Combining changes from one branch into another.

---

## Steps

### 1. Switch to main branch

```bash
git checkout main
```

---

### 2. Merge branch

```bash
git merge feature-login
```

---

## Example

```
main ← feature-login
```

---

## Summary

* Work in branch
* Merge into main
