# Common Git Mistakes

## Mistake 1: Committing Everything Together

Bad:

* Large unclear commits

Good:

* Small meaningful commits

---

## Mistake 2: Working on Main Branch

Always use feature branches.

---

## Mistake 3: Not Pulling Latest Code

Leads to conflicts.

---

## Mistake 4: Poor Commit Messages

Avoid:

* "update"
* "fix"

---

## Summary

Avoid mistakes to maintain clean workflow
