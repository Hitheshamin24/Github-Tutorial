# Open Source Contribution Guide

## What is Open Source?

Projects where anyone can contribute.

---

## Steps to Contribute

1. Find a repository
2. Fork it
3. Clone it
4. Create a branch
5. Make changes
6. Push code
7. Create Pull Request

---

## Tips

* Start with small issues
* Read contribution guidelines
* Write clean code

---

## Why Contribute?

* Gain experience
* Build portfolio
* Learn from real projects

---

## Summary

Open source = Real-world experience
