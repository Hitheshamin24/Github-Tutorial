# Introduction to GitHub Actions

## What is GitHub Actions?

GitHub Actions is a tool for **automation**.

---

## What Can It Do?

* Run tests
* Deploy applications
* Automate workflows

---

## Example Use Case

* Automatically run tests when code is pushed

---

## Basic Idea

```
Code Push → Workflow Trigger → Action Runs
```

---

## Summary

GitHub Actions = Automation inside GitHub
