# Git Pull vs Fetch

## Git Fetch

```bash
git fetch
```

* Downloads changes
* Does NOT merge

---

## Git Pull

```bash
git pull
```

* Downloads + merges changes

---

## Difference

| Command   | Action           |
| --------- | ---------------- |
| git fetch | Download only    |
| git pull  | Download + merge |

---

## Summary

Fetch = Safe update
Pull = Update + apply changes
