# What is GitHub?

## Definition

GitHub is a **cloud-based platform** used to host Git repositories.

---

## Simple Explanation

* Git → works on your local system
* GitHub → stores your code online

---

## Why Use GitHub?

* Backup your code
* Share projects
* Collaborate with others
* Build portfolio

---

## Key Features

* Repositories
* Pull Requests
* Issues
* Collaboration tools

---

## Git vs GitHub

| Git            | GitHub            |
| -------------- | ----------------- |
| Local tool     | Cloud platform    |
| Tracks changes | Hosts code        |
| Works offline  | Requires internet |

---

## Summary

GitHub = Online storage + collaboration for Git projects
