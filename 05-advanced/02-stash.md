# Git Stash

## What is Stash?

Stash temporarily saves your changes without committing.

---

## Use Case

You are working on something, but need to switch branches quickly.

---

## Commands

```bash
git stash
git stash pop
git stash list
```

---

## Example

1. Make changes
2. Run `git stash`
3. Switch branch
4. Return and use `git stash pop`

---

## Summary

Stash = Temporary storage of changes
